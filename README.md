
# AI Powered E-Commerce Store (Django)

This project was developed as part of an assignment to build an intelligent E-Commerce web application using Django and basic AI features.

The system allows users to browse products, place orders, and view order history. It also includes smart features such as product recommendations and AI-based search.

---

## Project Features

**User Features**
- User registration and login
- Browse available products
- Place product orders
- View order history

**Admin Features**
- Add and manage products
- Manage customers
- Track orders and order items

**AI Features**
- Smart product search
- Product recommendation system
- Price prediction logic

## How to Run the Project

1. Install required packages
    ```bash
    pip install -r requirements.txt
    ```

2. Apply migrations
    ```bash
    python manage.py migrate
    ```

3. Run the Django server
    ```bash
    python manage.py runserver
    ```

4. Open your browser and navigate to
    ```
    http://127.0.0.1:8000/
    ```

## Project Structure

- `ecommerce/` – Main Django project
- `store/` – Application with models, views, and logic
- `templates/` – HTML templates for UI
